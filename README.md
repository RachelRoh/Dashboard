# DUT System — 단말 관리 시스템

파트별 단말 장비의 재고, 대여, 폐기를 관리하는 웹 애플리케이션입니다.

## 페이지 안내

### 📊 단말 현황 요약
모델별 가용/대여중/미사용/고장/폐기 수량을 한눈에 확인합니다.

### 📱 모델별 현황
모델을 선택해 해당 단말 목록을 조회합니다.
소유팀, 소유자, 상태로 필터링하거나 키워드로 검색할 수 있습니다.

### 👥 파트별 현황
팀별로 단말 목록을 관리합니다.
- 단말 추가 / 일괄 추가 (엑셀 붙여넣기)
- 단말 편집 (모델, 시리얼번호, 소유자, 비고 등)
- 이관 / 미사용 / 고장 처리
- 폐기 예정 목록 확인

### 📒 대여 현황
단말 대여 등록 및 반납 처리를 합니다.
대여 이력을 팀별로 조회할 수 있습니다.

### 🗑️ 폐기 현황
미사용/고장 처리된 단말의 폐기를 확정하거나 폐기 대기로 되돌립니다.

### 👤 멤버 관리
소유자/대여자로 등록될 멤버를 추가하거나 삭제합니다.

---

## 단말 상태 흐름

```
가용 → 대여중        (대여 등록)
대여중 → 가용        (반납)
가용 → 가용          (이관 — 팀 변경)
가용 → 고장 / 미사용 (고장·미사용 처리)
고장 / 미사용 → 폐기 완료  (폐기 처리)
폐기 완료 → 미사용 / 고장  (폐기 취소)
```
